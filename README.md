## This is a extension for CodeLAB RP2040 board

## Input

### Buttons
- **B0** → `IO2`  
- **B1** → `IO3`

### Analog Inputs & Outputs
- `A0`, `A1`, `A2`, `A3`, `A4`, `A5`, `A6`, `A7`

---

## Output

### Buzzer
- **Buzzer** → `IO11`

### Motors (DRV8874 Driver)
- **M0**
  - `IO24`
  - `IO25`
- **M1**
  - `IO22`
  - `IO23`
- **M2**
  - `IO20`
  - `IO21`
- **M3**
  - `IO18`
  - `IO19`

### Servos
- **SV0** → `IO17`  
- **SV1** → `IO16`  
- **SV2** → `IO15`  
- **SV3** → `IO14`  
- **SV4** → `IO13`  
- **SV5** → `IO12`
